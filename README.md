# 💻 Calculadoras em JavaScript

Este repositório contém projetos desenvolvidos para prática de lógica de programação e cálculos do dia a dia.

---

## 📂 Projetos

### 🧮 Calculadora de INSS
Aplicação que realiza o cálculo do desconto do INSS com base no salário informado.

#### 🔹 Funcionalidades:
- Cálculo automático do INSS
- Exibição do salário com desconto
- Interface simples e intuitiva

#### 💻 Tecnologias:
- HTML
- CSS
- JavaScript

---

### ⚖️ Calculadora de IMC
Aplicação que calcula o Índice de Massa Corporal (IMC) e informa a classificação do usuário.

#### 🔹 Funcionalidades:
- Cálculo do IMC
- Classificação (Abaixo do peso, Normal, Sobrepeso, etc.)
- Entrada de peso e altura

#### 💻 Tecnologias:
- HTML
- CSS
- JavaScript

---

## 🚀 Objetivo

- Praticar lógica de programação
- Trabalhar com funções em JavaScript
- Criar aplicações úteis para o dia a dia
